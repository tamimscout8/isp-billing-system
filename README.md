# ISP Billing System

একটি সম্পূর্ণ ISP (Internet Service Provider) বিলিং সিস্টেম যা PHP, MySQL, HTML, CSS এবং JavaScript দিয়ে তৈরি।

## ফিচার

✅ **কাস্টমার ম্যানেজমেন্ট** - গ্রাহক রেজিস্ট্রেশন এবং প্রোফাইল ম্যানেজমেন্ট
✅ **প্ল্যান ম্যানেজমেন্ট** - বিভিন্ন ইন্টারনেট প্যাকেজ তৈরি এবং পরিচালনা
✅ **বিলিং সিস্টেম** - মাসিক বিল জেনারেশন এবং ট্র্যাকিং
✅ **পেমেন্ট প্রসেসিং** - অনলাইন পেমেন্ট এবং ট্রানজ্যাকশন ম্যানেজমেন্ট
✅ **ড্যাশবোর্ড/রিপোর্টস** - সম্পূর্ণ বিশ্লেষণ এবং রিপোর্টিং

## প্রযুক্তি

- **Backend**: PHP (খাঁটি)
- **Database**: MySQL
- **Frontend**: HTML5, CSS3, JavaScript (ভ্যানিলা)

## ইনস্টলেশন

1. রিপোজিটরি ক্লোন করুন
2. MySQL ডাটাবেস তৈরি করুন
3. `config/database.php` সেটআপ করুন
4. স্কিমা ইম্পোর্ট করুন
